# BLADE-BALL-REMAKE-HELP
Blade Ball remake scripts written for Roblox in Lua: BallSpawnerScript drives the dodge and parry gameplay loop, parryinput reads client-side parry timing, AntiShieldspawn strips forcefields from player characters, StarBurstVFX renders hit and shield visual effects, and Pingpongserver with Pingfpsdisplay together power a live FPS/ping HUD.
